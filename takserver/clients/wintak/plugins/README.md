# WinTAK Plugins

Place WinTAK plugin archives (.zip) in this directory.

## Recommended Plugins for Drone Defense Operations

| Plugin | Description | Source |
|--------|-------------|--------|
| **TAK Radar** | Radar track visualization | TAK Plugin Repository |
| **ADS-B Integration** | Display ADS-B (aircraft) data | Community plugin |
| **Drone ID Decoder** | Decode RemoteID drone signals | Community plugin |
| **MGRS Grid Overlay** | Military Grid Reference System overlay | Built-in |

## Installation

1. Download the plugin .zip file
2. In WinTAK: **Plugins > Load Plugin > Select .zip**
3. Restart WinTAK if required
